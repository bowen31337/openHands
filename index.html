openhands.sdk

The heart of the framework. It defines the core Agent, Conversation, and LLM abstractions. It manages the Event Stream, treating all interactions as immutable events. It is completely environment-agnostic.

Local to Remote in One Line

Prototype locally with zero friction. Deploy to production containers with a single configuration change. The agent logic remains exactly the same.

app.py (Local)
from openhands.sdk import LLM, Conversation from openhands.tools.preset.default import get_default_agent # No workspace import needed for local default # 1. Configure Model llm = LLM(model="anthropic/claude-sonnet-4.1", api_key="...") # 2. Create Agent agent = get_default_agent(llm=llm) # 3. Create Conversation (Local by default) conversation = Conversation( agent=agent, workspace="./my_project" # Runs on host filesystem ) conversation.send_message("Create a Hello World script.") conversation.run()
